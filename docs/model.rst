SepiaModel
==========

The SepiaModel object is created after the SepiaData object is created and manipulated (standardized, basis created).

The SepiaModelSetup function is used to create a SepiaModel object.

.. note:: SepiaModelSetup should always be used to create a SepiaModel object.
