.. _sepiamodel:

SepiaModel
==========

The SepiaModel object is created after the SepiaData object is created and manipulated (standardized, bases created).

The SepiaModelSetup function is used to create a SepiaModel object.

.. note:: SepiaModelSetup should always be used to create a SepiaModel object. Users can then modify components of the
    model object afterward calling SepiaModelSetup.
